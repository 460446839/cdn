# Page Optimization {#concept_ubt_4bm_xdb .concept}

## Introduction {#section_fjd_tbm_xdb .section}

The page optimization function can be used to delete comments and repeated whitespaces embedded in HTML in order to remove redundant page content, reduce file size, and improve the efficiency of delivery.

## Procedure {#section_r5j_5bm_xdb .section}

**Note:** When we are optimizing your page, the file's md5 value will be changed. It will be different from that of the file I your origin site. Do not enable this feature if your origin site has some verification.

1.  Go to Domain Namespage, select the domain name, then click **Manage**.
2.  ![](http://static-aliyun-doc.oss-cn-hangzhou.aliyuncs.com/assets/img/5160/15450383347303_en-US.png)
3.  Enable the function in **Performance Optimization** \> **Page Optimization**.
